
Description: Runs a hello-world Python script in the MSA validated environment. Displays
             a text message as an animated starfield graphic in a window.
             Used to verify that the Python validated environment is working correctly
             for the MSA module.

Usage: jrc_msa_py_hello "text"

Arguments:
       text              string between quotes. Should contain no special characters
       -h | --help       Show this help message

Example:
       jrc_msa_py_hello "Hello MSA"
