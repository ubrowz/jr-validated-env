
Description: Runs a hello-world Python script in the validated environment. Displays a
             text message and saves an animated GIF to the Downloads folder.
             Used to verify that the Python validated environment is working correctly.

Usage: jrc_py_hello "text"

Arguments:
       text              string between quotes. Should contain no special characters
       -h| --help        Show this help message

Example:
       jrc_py_hello "Hello World"
