Core

24 scripts

The foundation of the JR Validated Environment. 24 scripts covering the full analytical workflow — from planning a study and simulating test data, through statistical analysis and process capability reporting. Core scripts use only base R; no additional packages are required beyond what ships with the validated environment.

Sample Size Planning — 10 scripts Determine how many samples you need before you collect data. Covers continuous measurements, pass/fail attributes, equivalence testing, paired comparisons, fatigue and reliability studies, and measurement system qualification.
Data Analysis — 7 scripts Analyse measurement datasets: descriptive statistics, normality testing, outlier detection, process capability indices (Cp, Cpk), Weibull survival analysis, Bland-Altman agreement, and attribute pass/fail verification.
Generate Test Data — 5 scripts Simulate datasets with specified distributions for test planning, validation, and training. Supports normal, lognormal, Box-Cox, square-root-transformed, and uniform distributions.
Prepare Data — 2 scripts Convert non-standard input files (tab-delimited text, legacy formats) to the two-column CSV format required by all JR scripts.

DoE — Design of Experiments

2 scripts

Plan and analyse factorial experiments to understand which process factors drive your response — with far fewer runs than one-factor-at-a-time testing. The DoE module consists of two scripts that work in sequence: one to generate the run matrix, one to analyse the results.

jrc_doe_design — Design generation Generates a randomised run matrix for full-factorial and fractional-factorial designs, output as a printable lab sheet and a companion data CSV ready for results entry.
jrc_doe_analyse — Results analysis Reads the completed results CSV and produces an ANOVA table, main effect and interaction plots, and a factor significance summary identifying which factors and interactions explain meaningful variation.

MSA — Measurement System Analysis

5 scripts

Qualify your measurement systems before you rely on data. A measurement system that contributes significant variation to results will mask real process signals and undermine any downstream analysis. The five MSA scripts cover the full range of AIAG MSA Fourth Edition study types.

jrc_msa_gauge_rr — Crossed Gauge R&R ANOVA-based Gauge R&R for the standard crossed design where every operator measures every part. Decomposes variation into repeatability, reproducibility, operator, and part-to-part components.
jrc_msa_nested_grr — Nested Gauge R&R Gauge R&R for destructive tests where parts cannot be re-measured by multiple operators. Uses a nested (hierarchical) ANOVA model.
jrc_msa_type1 — Type 1 Repeatability Study Single-operator, single-part study to assess instrument repeatability before a full Gauge R&R. Reports Cg and Cgk against the tolerance.
jrc_msa_linearity_bias — Linearity & Bias Evaluates whether measurement error varies across the measurement range and whether there is a systematic offset relative to a reference standard.
jrc_msa_attribute — Attribute Agreement Analysis Assesses the consistency of go/no-go gauges, visual inspection, or any categorical measurement system. Reports within-appraiser, between-appraiser, and appraiser-vs-reference agreement with Kappa statistics.

SPC — Statistical Process Control

5 scripts

Monitor production processes with control charts and detect assignable causes of variation before they produce nonconforming product. All SPC scripts apply Western Electric rules for out-of-control detection and produce annotated charts flagging any violations.

jrc_spc_imr — Individuals & Moving Range chart For individual measurements taken one at a time. The most common chart for low-volume production or slow processes where subgrouping is not practical.
jrc_spc_xbar_r — Xbar-R chart For subgroup averages and ranges. Standard choice for subgroup sizes of 2–8 where the range is a stable estimator of within-subgroup spread.
jrc_spc_xbar_s — Xbar-S chart For subgroup averages and standard deviations. Preferred for larger subgroup sizes (≥9) where the standard deviation is more efficient than the range.
jrc_spc_p — p-chart For the proportion of nonconforming units in a subgroup. Handles variable subgroup sizes and adjusts control limits accordingly.
jrc_spc_c — c-chart For counts of nonconformities (defects) per constant inspection unit.

AS — Acceptance Sampling

4 scripts

Design and evaluate statistical sampling plans for lot inspection decisions. The AS module supports both attribute sampling (count of defectives) and variables sampling (measurements against a specification limit), aligned with ISO 2859 and ISO 3951.

jrc_as_attributes — Attribute sampling plan design Designs single and double attribute sampling plans to meet specified AQL and RQL (LTPD) requirements. Reports sample sizes, acceptance numbers, ASN, and a full OC curve table.
jrc_as_variables — Variables sampling plan design Designs a variables sampling plan for normally distributed measurements using the k-method. Typically achieves the same protection as an attribute plan with a substantially smaller sample size.
jrc_as_oc_curve — OC curve analysis Generates an Operating Characteristic curve for any attribute plan (n, c) across a user-specified range of lot fraction defective. Identifies the AQL and RQL operating points.
jrc_as_evaluate — Lot evaluation Evaluates a sampled lot against an existing plan — in attributes mode (count of defectives vs. acceptance number) or variables mode (sample mean and standard deviation vs. k-value). Returns a clear ACCEPT or REJECT decision.
